Untuk membuka tampilan website dibutuhkan XAMPP versi manapun

-run xampp dan run apache dan mysql
-buka browser dan run localhost/phpmyadmin di search bar
-buat database baru bernama CampusDB dan import file CampusDB.sql didalam file rar
-copy folder didalam rar dan paste dalam folder c:/xampp/htdocs/
-buka web dengan menjalankan command localhost/WebCampus dalam searchbar browser
